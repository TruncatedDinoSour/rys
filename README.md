<h1 align="center">Rys programming language</h1>

<p align="center">
    <i>A reverse polish notation, stack based, concatinative, compiled programming language</i>
</p>

## Requirements

-   `ld` linker
-   `nasm` assembly compiler
-   `python3.8` and up
-   Python `typing`
-   Linux
